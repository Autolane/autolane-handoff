# Autolane Handoff Dashboard

Next.js web dashboard for visualizing the two-stage inference pipeline.

## Features

- Video player with canvas overlay for Grounding DINO BBoxes
- Chain-of-thought reasoning panel
- Agent status board
- Safety score gauge
- Handoff sequence timeline
- Alert feed

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
